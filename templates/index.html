Applying, Interpreting, and Comparing Machine Learning Models on the WDBC Dataset.

Data Science Objective

The primary goal of this project is a binary classification task: determining whether a given breast mass record is Benign or Malignant.

This determination is based on the features available in the public Wisconsin Diagnostic Breast Cancer (WDBC) dataset.

✓
The classification target, or label, is the attribute named "diagnosis". All other attributes used for prediction are numeric.

Project Scope and Methodology

⚙ Model Replication

The core work involves replicating the results from an existing research article. This requires training, evaluating, and presenting a comparison between 6 distinct Machine Learning algorithms.

★ Fair Comparison

A critical step for robust comparison is ensuring all 6 models are trained using the same hyperparameters, training procedures, and dataset splits to guarantee an apples-to-apples comparison.

💮 Extra Initiatives

Beyond replication, this project proposes future data science objectives and initiatives, such as exploring advanced feature engineering, trying ensemble methods, or conducting deeper interpretability analyses to improve upon the established baseline.
